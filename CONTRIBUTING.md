## How Can I Contribute?

### Reporting Bugs
- Use the GitHub Issues tracker to report bugs.
- Include a clear title and description.
- Provide as much relevant relevant information as possible, including steps to reproduce.

### Running Tests
Before submitting a PR, please run the integrity test:
```bash
python tools/test_integrity.py
```

### Suggesting Enhancements
- Enhancement suggestions are tracked as GitHub issues.
- Explain why the enhancement would be useful to most users.

### Pull Requests
- Fill in the required template (if any).
- Ensure your code follows the existing style of the project.
- Make sure your changes do not break existing functionality.
